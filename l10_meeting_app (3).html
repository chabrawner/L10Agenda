First time setup? Follow the instructions in the setup guide below, then paste your credentials here.

📋 Setup Guide (click to expand)

Step 1 — Create a free Supabase account
Go to supabase.com and sign up for free. Create a new project (any name, any region).

Step 2 — Create the database table
In your Supabase project, click SQL Editor in the left sidebar and run this query:

create table l10_store (
  key text primary key,
  value text not null
);

alter table l10_store enable row level security;

create policy "Allow all" on l10_store
  for all using (true) with check (true);

Step 3 — Get your credentials
Go to Project Settings → API. Copy your Project URL and the anon public key. Paste them in the fields above.

Step 4 — Host the app (optional but recommended)
Go to netlify.com, sign up free, and drag-and-drop this HTML file onto the dashboard. You'll get a shareable URL your whole team can bookmark.

That's it! Your credentials are saved in this browser. Anyone else who opens the app just needs to enter the same URL and key once.
